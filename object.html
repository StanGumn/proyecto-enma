La desestructuración nos permite extraer valores de arrays y objetos (incluso profundamente anidados) y almacenarlos en variables con una sintaxis más conveniente.

Destruyendo arreglos 
        var arr = [1, 2, 3, 4];
        var a = arr[0];
        var b = arr[1];
        var c = arr[2];
        var d = arr[3];
    
        let [a, b, c, d] = [1, 2, 3, 4];

        console.log(a); // 1
        console.log(b); // 2

Destruyendo objetos 
        var luke = { occupation: 'jedi', father: 'anakin' };
        var occupation = luke.occupation; // 'jedi'
        var father = luke.father; // 'anakin'
        
        
        let luke = { occupation: 'jedi', father: 'anakin' };
        let {occupation, father} = luke;

        console.log(occupation); // 'jedi'
        console.log(father); // 'anakin'
